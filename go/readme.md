## Switch

## –°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ —Å–ª–∞–π—Å–æ–≤

–°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –ø–∞–∫–µ—Ç `sort` –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç —Ñ—É–Ω–∫—Ü–∏–∏ –¥–ª—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏:

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —Ñ—É–Ω–∫—Ü–∏—é `Slice(x interface{}, less func(i, j int) bool)`. –°–ª–µ–¥—É–µ—Ç –∑–∞–ø–æ–º–Ω–∏—Ç—å, —á—Ç–æ –ø—É—Å—Ç–æ–π –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å `interface{}` –≤ Go –æ–∑–Ω–∞—á–∞–µ—Ç —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö, –ø–æ–¥ –∫–æ—Ç–æ—Ä—ã–π –ø–æ–¥—Ö–æ–¥–∏—Ç –ª—é–±–æ–π –¥—Ä—É–≥–æ–π —Ç–∏–ø.

–í–Ω—É—Ç—Ä–∏ —Ñ—É–Ω–∫—Ü–∏–∏ **sort.Slice** –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –±—ã—Å—Ç—Ä–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞. –í –ø–∞–∫–µ—Ç–µ —Ç–∞–∫–∂–µ –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –≤—Å—Ç–∞–≤–∫–∞–º–∏ `sort.SliceStable`:
